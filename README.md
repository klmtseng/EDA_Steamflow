# EDA_Steamflow

This repository contains a simple Python utility for performing basic
Exploratory Data Analysis (EDA) on arbitrary datasets. The tool was
inspired by popular EDA notebooks such as the Titanic analysis on Kaggle.

## Requirements

- Python 3
- pandas
- seaborn
- matplotlib
- tkinter (usually included with standard Python installations)

## Usage

Run the script and select a data file (CSV, Excel, or JSON) from the file
selection window:

```bash
python eda_tool.py
```

The script displays dataset information, summary statistics, missing value
counts and basic visualisations. The file selection window allows you to
browse and choose any supported data file on your system.
